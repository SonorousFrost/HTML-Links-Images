Absolute and Relative Links

Absolute links - Link to pages on other sites. Made up of "protocol://domain/path" an absolute link will always contain the protocol and domain of the destination.

Relative Links - Link to other pages within a site, They do not include domain name. Assuming the pages will have the same name. Relative paths only include the file path to the other page relative to the page the link is created on.

Img element

Besides the src attribute every img elemnt should have an alt attribute. Used to describe the image in the event of a loading failure.
